# Aasir Farrukh - Portfolio

A modern, responsive personal portfolio website built with React and Tailwind CSS.

## Features

- **Dark/Light Mode** - Toggle between themes with smooth transitions
- **Responsive Design** - Optimized for all screen sizes
- **Interactive UI** - Parallax effects, animations, and smooth scrolling
- **Project Showcase** - Image gallery modals with keyboard navigation
- **Certificate Viewer** - Full-size certificate modal display

## Tech Stack

- **React 18** - UI framework
- **Tailwind CSS** - Utility-first styling
- **Lucide React** - Icon library


## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
```

## Sections

- **Hero** - Introduction with animated background
- **About** - Personal bio and summary
- **Education** - Academic background
- **Courses** - Online courses completed
- **Certifications** - Professional certificates
- **Skills** - Technical skills showcase
- **Experience** - Work history
- **Projects** - Portfolio of work with demos
- **Contact** - Contact information
